# Shadow 项目上下文

这个仓库是一个预备做成 HOI4 工作站的项目。

## 项目定位

- 主程序负责加载插件，并提供基础功能。
- 插件负责提供更丰富、可扩展的功能。
- 每一个插件都是和主程序位于同一目录层级下的独立项目。
- 主程序和插件之间应保持清晰边界，避免把插件专属功能写死在主程序中。

## 技术与界面约定

- 项目采用 Avalonia。
- UI 采用 Fluent 设计风格。
- 已安装 `FluentAvaloniaUI` 软件包，开发界面时优先使用现有 FluentAvaloniaUI 能力和项目已有样式。

## 协作约定

- 可以调用 MCP 工具辅助查看、分析和修改项目。
- 切换聊天窗口后，优先阅读本文件来恢复项目背景。
