# Shadow 1.0.0-beta.1

Shadow `1.0.0-beta.1` is the first public beta of the HOI4 workstation shell.
It includes the desktop host, the plugin abstraction project, and the bundled
HOI4 Launcher plugin.

## Highlights

- Avalonia desktop host with Fluent styling.
- Runtime plugin discovery from the application `Plugins` directory.
- HOI4 Launcher plugin for mod discovery, playset management, DLC selection,
  game settings, and game launch.
- Paradox Launcher playset import as read-only playsets.
- Shared playset storage and mod index export under
  `%APPDATA%\Posdaca\Hoi4Workspace`.
- Command-line plugin dispatch with `hoi4.launch`.

## Download

Use `Shadow-1.0.0-beta.1-win-x64.zip` on Windows x64. Extract it to a writable
folder and run `Shadow.exe`.

## Beta Notes

- Back up the HOI4 user directory before testing.
- The release writes HOI4 launcher state such as `dlc_load.json`.
- Plugin APIs and workspace schemas may change before the stable release.

## 中文说明

Shadow `1.0.0-beta.1` 是 HOI4 工作站外壳的第一个公开 beta 版，包含桌面主程
序、插件抽象项目，以及内置的 HOI4 启动器插件。

### 重点

- 使用 Avalonia 和 Fluent 风格的桌面主程序。
- 从应用程序 `Plugins` 目录运行时发现插件。
- HOI4 启动器插件支持 Mod 发现、播放集管理、DLC 选择、游戏设置和启动游戏。
- 可将 Paradox Launcher 播放集导入为只读播放集。
- 播放集和 Mod 索引存储在 `%APPDATA%\Posdaca\Hoi4Workspace`。
- 支持命令行插件命令分发，包含 `hoi4.launch`。

### 下载

Windows x64 用户下载 `Shadow-1.0.0-beta.1-win-x64.zip`，解压到可写目录后运行
`Shadow.exe`。

### Beta 提醒

- 测试前请先备份 HOI4 用户目录。
- 该版本会写入 HOI4 启动状态文件，例如 `dlc_load.json`。
- 插件 API 和工作区 schema 在稳定版前仍可能变化。
